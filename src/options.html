Bench Calculator v1.01

Made by Lennart Coopmans

Simple calculator, bc style, as Opera extension.

A small calculator that doesn't require you to click on buttons. It's somewhat similar to the gnu bc command line calculator. Simply type a mathematical expression and press enter.

Supported operators and functions:

Arithmetic
  • +
  • -
  • *
  • /
  • %
  • ^
Logical
  • !
  • &
  • |
Comparison
  • ==
  • !=
  • >
  • <
  • >=
  • <=
  • <>
Functions
  • AVG
  • ABS
  • ACOS
  • ASC
  • ASIN
  • ATAN
  • CDATE
  • CHR
  • COS
  • DATE
  • FIX
  • HEX
  • IIF
  • LCASE
  • LEFT
  • LOG
  • MAX
  • MID
  • MIN
  • RAND
  • RIGHT
  • ROUND
  • SIN
  • SQRT
  • TAN
  • UCASE

Setting variables

You can define variables by entering the following line:

<variable name> = <expression>
The expression will be evaluated, and the result stored in the variable.

Clear

Will clear the history. Command can be triggered by typing clear or reset
